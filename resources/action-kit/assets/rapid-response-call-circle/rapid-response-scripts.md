# Rapid Response Scripts

## Informational Script
```
Hey [Name], it’s [Caller] with [Org]. I’m checking in with a quick briefing about [situation].

1. What happened: [two-sentence summary].
2. What we need: stay ready for an action within the next [time window].
3. We’ll text you by [time] with final details.

Are you available if we move ahead? (Mark response in the log.)
```

## Mobilization Script
```
Hi [Name], [Caller] with [Org]. We need you at [location] at [time] for [action].

- Wear / bring: [items]
- Point person on-site: [name + phone]
- If you can’t attend, can you call [target] using this script: [link]?

Can I count on you to be there?
```

## Disposition Codes
- READY – confirmed attendance
- INFO – needs more details
- LATER – can’t do this action but wants follow-up
- NO – unavailable
