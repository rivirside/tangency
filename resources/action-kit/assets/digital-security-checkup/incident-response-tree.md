# Incident Response Tree

## If you suspect an account breach
1. Notify security lead immediately.
2. Freeze access / change passwords.
3. Document what you know.
4. Contact:
   - Legal: [name/phone]
   - Comms: [name/phone]
   - Tech ally: [name/phone]

## If devices are seized/confiscated
1. Do not unlock without counsel.
2. Call legal line.
3. Inform rapid-response team for support.

## If harassment/doxxing occurs
1. Screenshot evidence.
2. Alert comms + care team.
3. Offer safety buddy + plan.
